# Cases

The category taxonomy, the coverage matrix, and one page per case live in
**[cases/README.md](cases/README.md)**: **8 categories** (cartography, semantic, joinability, correlation,
geographic, temporal, quality, affinity) across **11 cases**. Each case carries a category, a `render_kind`, a
variant family (a real parametric knob), and is built as a read over the mined knowledge graph. The App shows one
case; Experiments and Benchmark summarize across categories.
